## git fetch

**git fetch** - связывается с удалённым репозиторием и забирает из него все изменения, которых у вас пока нет и сохраняет их локально.

```bash=
git fetch [remote-name]
```

Данная команда связывается с указанным удалённым проектом и забирает все те данные проекта, которых у вас ещё нет. После того как вы выполнили команду, у вас должны появиться ссылки на все ветки из этого удалённого проекта, которые вы можете просмотреть или слить в любой момент.

Важно отметить, что команда _git fetch_ забирает данные в ваш локальный репозиторий, но не сливает их с какими-либо вашими наработками и не модифицирует то, над чем вы работаете в данный момент. Вам необходимо вручную слить эти данные с вашими, когда вы будете готовы.

+  _origin_ - извлекает все наработки, отправленные на этот сервер после того, как вы его клонировали (или получили изменения с помощью _fetch_).

```bash=
git fetch origin
```
